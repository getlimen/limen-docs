## Long-form (for blog / GitHub Release notes / HN body)

### The short version

We're releasing [**Limen v0.1.0**](https://github.com/getlimen/limen) — a self-hosted infrastructure platform. It's three tools in one control plane:

1. **Docker deploy management** — register a registry image, Limen polls for new digests, rolls your containers forward with health-check + auto-rollback
2. **Reverse proxy with automatic TLS** — expose any service on a public hostname; Let's Encrypt certs just happen
3. **WireGuard hub-and-spoke VPN** — remote hosts dial home over WireGuard; no inbound ports to open

Install it with `docker compose up`, log in via OIDC, and enroll the host you want to manage. That's the workflow.

### Who it's for

Admins running infrastructure across 1–30 hosts. Homelab, small business, solo developer. You don't need Kubernetes, you don't want Coolify+Pangolin+Traefik as three separate installs, and you don't want to pay for a commercial option.

### What's in v0.1.0

- Single-host or multi-host deployment (role flags: `control`, `proxy`, `docker`)
- ASP.NET Core 10 manager + Angular 21 UI, PostgreSQL as the sole datastore
- `Ostiarius` — custom reverse proxy on YARP with `LettuceEncrypt-Archon` for ACME
- `Forculus` — WireGuard hub wrapping the kernel `wg` tool
- `Limentinus` — universal agent with embedded `wireguard-go` for userspace tunnels
- Resource-level authentication (password, SSO via OIDC, email allowlist with magic-link)
- Deployment queue with persistent status, real-time log streaming, automatic rollback on health-check failure
- Single admin account (multi-admin + RBAC coming in v0.2.0)

### What's not in v0.1.0

Deferred to future releases:
- **v0.2.0** — multi-admin + RBAC, CLI (`limen-cli`)
- **v0.3.0** — Kubernetes / k3s support (new `k8s` node role)
- **v0.4.0** — registry webhooks, blueprints, cert-expiry alerts
- **v0.5.0** — PIN / WebAuthn / API token auth modes

See the full [roadmap](https://github.com/getlimen/limen/blob/main/docs/roadmap.md).

### Try it

```bash
git clone https://github.com/getlimen/limen
cd limen
cp template.env .env
# edit .env — set POSTGRES_PASSWORD, OIDC_*
docker compose up -d
```
